# PDF Compressor

App web para comprimir PDFs escaneados. Construida con Streamlit + pdf2image.

## Correrla localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Subir a Streamlit Cloud (gratis)

1. Crear cuenta en https://github.com y subir esta carpeta como repositorio
2. Ir a https://streamlit.io/cloud y conectar tu cuenta de GitHub
3. Seleccionar el repositorio y el archivo `app.py`
4. Click en **Deploy** — en 2 minutos tienes tu link listo

> Nota: Streamlit Cloud instala poppler automáticamente desde `packages.txt`
